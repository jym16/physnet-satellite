# Physical Networks Satellite Archive

This repository contains the public website for the **Physical Networks** satellite series at NetSci.

The site is built with [Quarto](https://quarto.org/) and serves two purposes:

1. Host the current edition of the satellite.
2. Preserve static archive pages for previous editions.

## Contents
